# MultiBot
An all in one Discord bot that has multiple features like WHOIS, AI generation of images, Encoding/Decoding and API-related features and commands. (they're gonna be added later the bot is still work in progress lol)
# FAQ (Frequently Asked Questions)
Q: How do I set this up?
A: To set up MultiBot you need to do these following steps.
First, Use git clone to clone this repo or download the ZIP file.
Secondly, when you download it rename the .env.example file to .env
Third, open it in vim/nano or whatever editor you use and put your own bot token and command prefix.
Q: What packages do I need for this to work?
A: You need the following: `python-dotenv pybalt pyyaml aiofiles discord.py`.
